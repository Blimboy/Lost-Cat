Lost-Cat is a free QR code generator for Metal Dog Tags.
<br/><br/>
https://blimboy.github.io/Lost-Cat/index
<br/><br/>
When the QR Code is scanned on a mobile phone, it will the provide the recipient with the Cat's name and contact phonenumber for the owner.
<br/><br/>
The recipient can then either call the phonenumber or click the button to send a text message to the cat owner with their current GPS location information.
<br/><br/>
Contents:
<br/>
* index = main web page (landing Page).
* QR = QR code generator
* index1 = web link including 1 phonenumber and cat name
